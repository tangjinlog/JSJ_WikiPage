## 실행 커맨드

```bash
# clone
git clone https://github.com/tangjinlog/JSJ_WikiPage.git

cd JSJ_WikiPage

yarn

# Node.js version >= v18.17.0 is required
yarn dev

# 기본 서버 실행
yarn server
```
